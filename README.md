- **Version** : Java 17.0.9
- **IDE** : IntelliJ

#
주요기능
- Miain클래스<br>
  -게임실행<br>
  -게임기록<br>
  -게임종료<br>


- BaseballGame클래스<br>
  -난이도 조절 (입력받은 수만큼의 길이)<br>
  -스트라이크,볼,정답확인<br>

- GameRecordManager클래스<br>
  -각 판당 시도 횟수 기록<br>
